# 🐾 AdoptApp - Backend

Backend desarrollado con Spring Boot para la plataforma AdoptApp.

## Tecnologías

- Java 21
- Spring Boot
- Spring Security
- PostgreSQL
- JWT
- WebSocket
- AWS

## Requisitos

- JDK 21
- Maven

## Ejecución

```bash
git clone <URL_DEL_REPOSITORIO>
cd AdoptApp-Backend

mvn clean install

mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

La aplicación estará disponible en:

http://localhost:8082

## Entidades principales

- Users
- Pets
- Hostels
- Adoptions
